<h2>Setting up a simple local web server</h2>

1. Install Python
2. Go to directory of the project you wish to serve
3. Excute 
`python -m http.server [<portNo>]`
for python 3.x or
`python -m SimpleHTTPServer [<portNo>]`
for python 2.x
  * The default port number is
 `8000`
